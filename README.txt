ZMachine Interpreter for IBM 3270 compatible terminals
======================================================

Small dongle that provides Z-Machine interpreter (Infocom games), TN3270 (Telnet for 3270), SSH and Wikipedia clients.

Plugs into the serial port and uses SDLC and SNA networking protocols to communicate with the terminal over a 25-pin RS232 port.

Specificly this was designed for an Informer 270 376/SNA terminal.
